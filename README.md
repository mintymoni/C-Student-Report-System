# Student Grade Calculator (C++)
This simple C++ program allows users to input a student's name, midterm grade, and final grade, and then computes the average.  
It also assigns a rank such as "With Honors" or "Failed" based on the computed average.

## Features
- Accepts user input (name, midterm, and final grades)
- Allows basic math operations (+, -, *, /)
- Calculates the average grade
- Displays the student's rank:

1. Clone or download this repository.
